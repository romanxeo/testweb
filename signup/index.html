SIGNUP Gatsby supports TypeScript by default

This means that you can create and write .ts/.tsx files for your pages, components, and gatsby-* configuration files (for example gatsby-config.ts).

For type checking youll want to install typescript via npm and run tsc --init to create a tsconfig file.
